Eraser Mapper

Usage

Go to Settings → Keyboard Shortcuts. Find Eraser Mapper in the list and replace your current Freehand brush tool and Set eraser mode shortcuts with the ones for the plugin.

Go to Settings → Scripts → Eraser Mapper.

Assign a brush preset to use as the eraser.

Select which tools you'd like to switch to that eraser upon hitting the eraser key (overriding the default transparency mask behavior).

Select whether you'd like to be able to hit the eraser key again while in erasing mode to switch back to drawing mode.
